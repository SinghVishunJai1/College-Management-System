# 🎓 College Management System

This is a College Management System built using Java, Hibernate, and MySQL. It helps manage students, faculty, library books, and courses in an organized way.

## ✨ Main Features
* **Organized Code:** Uses DAO and Service layers to keep the code clean.
* **Hibernate:** Uses Hibernate to connect and manage the database easily.
* **Modules:** Different folders for Students, Library, Faculty, and Courses.

## 🛠️ Tools Used
* **Language:** Java
* **Framework:** Hibernate
* **Database:** MySQL
* **Build Tool:** Maven

## 📂 How the code is organized
* **`dao/`**: Handles database actions.
* **`model/`**: Defines data structures (like Student, Faculty).
* **`service/`**: Handles the main logic of the application.
* **`util/`**: Contains configuration settings.
